## M5 Phase 3 keyboard completion

## M5 Phase 4 catalog, consistency, and keyboard legend

CODEX2 completed `td-f1fb95` and the adopted legend item `td-40102b` from spec

bindings, help copy, and required surfaces. Its exact 24-ID inventory includes

The View menu toggles the macOS-only Keyboard Legend with Option-Command-L.

retaining no shortcut hints, and empty-area right-click adds Expand All and
Collapse All. iOS Help has no hardware-keyboard material and again documents
the touch keyboard accessory, including the keyboard-down dismiss button.

`docs/m5-parity-matrix.md` records every required and forbidden surface for the
integrated Gate M4 audit. Because Phase 4 added source and test files,
`xcodegen generate` regenerated `Listsurf.xcodeproj`. Automated verification:

stop-and-report condition was encountered; the Window-scene open/dismiss path
