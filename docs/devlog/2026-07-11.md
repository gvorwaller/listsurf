## What shipped

- **OPML codec** (M3-1): SAX decoder tolerant of real-world OPML — verified against two real CarbonFin exports (`docs/fixtures/`), which confirmed `_note`, `_status='checked'/'indeterminate'/absent`, OPML 1.0 with single-quoted attributes. Encoder writes `&#10;` so multi-line notes survive attribute normalization (CarbonFin's own encoder loses them — ours doesn't).
- **Markdown exporter** (M3-2): D6 format — heading, notes, `- [ ]`/`- [x]`, 2-space depth indent, `×N` quantities, indented note continuations.
- **Additive ImportPlanner + lenient validation** (M3-3): every UUID reminted (file IDs never persist), parent remap, missing-parent/cycle repair with counts, `ParentValidationPolicy` keeping replace-all strict.
- **Insert-only `addListsAndItems`** (M3-4): one transaction, rollback, and a mandatory collision preflight — required because `mergeByPropertyObjectTrump` silently upserts on ID collision instead of throwing.
- **AppStore layer** (M3-5): per-list JSON/OPML/Markdown exports on drained snapshots; prepare/commit additive import with BOM-tolerant byte-level sniffing; "(Imported N)" title resolution; path-bearing `DecodingError` messages for the paste-back-to-LLM loop; drain retrofitted into replace-all import.
- **UI wiring** (M3-6): Import List in sidebar/menus/empty state; per-list export + Markdown share entries in the single list action menu; dual-mode file importer; accept/discard import summary sheet; Markdown preview sheet with Copy + ShareLink; OPML UTType declaration; File menu item.
- **Diagnostics** (M3-7): Settings → Data with live counts, store size (metadata only — never opens the SQLite file), selectable store path, Reveal in Finder (macOS), Last Export.
- **Help** (M3-8): Import & Export section, replace-vs-add clarification, and the shipped LLM prompt with Copy button — the manual-LLM import workflow is now a documented product feature.
- **Ride-along fix** (td-6d881a): InspectorView navigationTitles removed — they leaked through `.inspector()` on iPhone and showed "List Info" instead of the list name (latent since the inspector was introduced).
